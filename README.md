# My DevOps Project

## Overview
A comprehensive DevOps project demonstrating Git version 
control and CI/CD pipeline automation using GitHub Actions.

## Technologies Used
- Git & GitHub
- GitHub Actions (CI/CD)
- Linux/Bash
- YAML

## Features
- Version controlled codebase
- Automated CI/CD pipeline
- Triggers automatically on every push
- Multi-step pipeline with build and test stages

## CI/CD Pipeline
The pipeline automatically runs when code is pushed:
1. Checks out code
2. Prints build information
3. Shows current date
4. Lists repository files
5. Confirms successful deployment

## Skills Demonstrated
- Git branching and commits
- GitHub Actions workflow
- YAML pipeline configuration
- DevOps automation
